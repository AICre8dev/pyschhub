# Build Plan

- A-Level Psychology resource hub for students and teachers
- Students can browse topics, access AI-generated study materials, and practice questions
- Teachers can create lesson plans, access teaching resources, and generate worksheets
- AI-powered features: essay marking, quiz generation, flashcard creation
- Topic browser covering all major A-Level Psychology specifications (AQA/OCR/Edexcel)
- Dashboard with role selection (Student/Teacher) for personalized experience
- Resource library with downloadable materials and external links
